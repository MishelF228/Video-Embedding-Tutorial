How to Embed a YouTube Video

This tutorial shows how to embed a YouTube video. I chose a cute kitten video for example.

Step 1: Go to YouTube

Open YouTube and find the video

Step 2: Click "Share"

Below the video, click the Share button.

Step 3: Select "Embed"

In the popup, click the Embed option to get the iframe code.

Step 4: Copy the iframe code

Copy the code starting with <iframe>.

Step 5: Paste code in your site

Paste the iframe code into your HTML where you want the video.
